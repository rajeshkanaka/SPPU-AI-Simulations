The Wumpus World is a classic AI problem — a 4x4 grid of 16 rooms where an intelligent agent must navigate safely to find gold. It demonstrates logical reasoning and decision making under uncertainty.

Everyday Analogy

Imagine you're exploring a dark cave with a torch. You can only see the room you're in. You smell bad odors, feel breezes, and see glitter — and from these clues, you must figure out where the monster and traps are, grab the treasure, and escape alive!

Key idea: The agent cannot see the full world. It must use percepts (sensory clues) and logical inference to decide safe moves — exactly what a Knowledge-Based Agent does.

Think of it this way

The agent is like a blindfolded person in a room — they can smell (stench), feel the air (breeze), see sparkle (glitter), touch walls (bump), and hear screams (scream). From these 5 senses, they must figure out the entire cave!

The agent uses a Knowledge Base (KB) — a collection of facts. At each step:

  1. Perceive: Agent enters a room and receives percepts (stench, breeze, etc.)
  2. TELL: Agent adds new facts to the KB — "Room (2,1) has no stench and no breeze"
  3. Infer: KB uses logical reasoning — "No stench at (2,1) means Wumpus is NOT at (1,1), (2,2), or (3,1)"
  4. ASK: Agent queries KB — "Which adjacent rooms are safe to visit?"
  5. Act: Agent moves to a safe room, and the cycle repeats
Example Inference

If room (1,2) has Stench but room (2,1) has no Stench, the agent can infer: Wumpus is at (1,3) — because (1,3) is adjacent to (1,2) but NOT adjacent to (2,1).

Why Single Agent?

The Wumpus doesn't make decisions or move — it's a static hazard like a pit. Only our intelligent agent reasons and acts. So this is a single-agent environment.
